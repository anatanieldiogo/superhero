# superhero
Superhero é uma mini sistema para a obtenção de informações dos seus superheros favoritos e ainda realizar uma batalha entre eles com base em valores vindo de uma API.

## Tecnologias
- HTML
- CSS
- JAVASCRIPT
- JQUERY
- AJAX
